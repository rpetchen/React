The project is an application which allows a user to search for books and manage books between three different 'shelves': Read, Currently Reading, and Want to Read. The Read shelf is for books the user of the application has already read, Currently Reading is for books the user is Currently Reading, and the Want to Read shelf is for books the user wants to read in the future.

The user can search for books and add them to any of the three shelves, and the users reuslts are persited allowing a user to manage their books going forward, placing them between the three shelves as they see fit.

This application is run on React, and in order to install all dependencies identified package.json file the the user will need to run 'NPM Install' form the command line. 

Once all the dependencies have been installed the user can start the application from the command line by running 'NPM Start'
